## 🛠 How to Trade

1. Click a buy/sell above, or [open a new issue](../../issues/new).
2. Title it **exactly**:
   - `market: BUY` — buy 1 share
   - `market: SELL <quantity>` — sell shares, e.g. `market: SELL 3`
3. A GitHub Action processes the order within seconds, updates this README, comments the result on your issue, and auto-closes it.

**Rules**
- Max **3 buys per 24-hour rolling window** per user.
- You can't sell more shares than you own.
- Price floor is **$1.00** — it never goes lower.
